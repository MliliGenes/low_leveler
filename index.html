### 1. Memory Management Fundamentals

- Intermediate Representation (IR)
- Code generation (to assembly)
- Compiler optimization levels and techniques (inlining, loop unrolling)

### 4. Assembly and Machine Code
- CPU architecture basics (x86-64 vs. ARM)
- Instruction Set Architecture (ISA)

- Assembly language syntax (AT&T vs. Intel)
- Linking and loading processes
- Debugging at the assembly level (GDB)

### 5. Operating System Interface
- System calls (syscalls) and kernel mode vs. user mode
- Process memory management and virtual address spaces
- Memory mapping (mmap) and protection (read/write/execute bits)
- Inter-Process Communication (IPC)
- File system I/O and buffering
- Concurrency: threading vs. multiprocessing, synchronization primitives

### 6. Performance and Optimization
- Cache hierarchy (L1, L2, L3), cache lines, and locality of reference
- Branch prediction and speculative execution
- Data-Oriented Design vs. Object-Oriented Programming
- Profiling tools (gprof, Valgrind, Perf)
- Understanding compiler optimization reports
- SIMD (Single Instruction, Multiple Data) instructions
